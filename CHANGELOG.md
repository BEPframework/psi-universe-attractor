# Changelog

All notable changes to BEP Field Lab are documented here.

## [4.0.0] — 2026-03

### Added
- **Fourth Field module** — upload two datasets (human agent + AI agent), configure AI evidence weight (α) and field effects (fE/fB), compute Fourth Field metrics: Errfield, Polfield, Courage trajectory, and regime classification. Grounded in Sections 5–6 of the BEP Field Theory of Conscious Systems preprint.
- **Bifurcation Explorer** — interactive bifurcation diagram showing how field dynamics change across parameter ranges.
- **Enhanced export** — download computed field scores, Fourth Field metrics, and bifurcation data as CSV.

### Changed
- Improved data mapping UI for proxy variable selection.
- Refined composite field score computation with baseline comparison.

## [3.0.0] — 2026

### Added
- CSV and Excel file upload support.
- Proxy variable mapping for Energy, Belonging, and Prediction fields.
- Composite field score computation with single-variable baseline comparison.
- Exploratory interpretation engine grounded in BEP theoretical framework.

## [2.0.0] — 2026

### Added
- Interactive field dynamics visualization.
- Dyadic human–AI dynamics exploration module.

## [1.0.0] — 2026

### Added
- Initial release of BEP Field Lab.
- Browser-based research tool for Belonging–Energy–Prediction field dynamics.
- No installation, server, or login required.

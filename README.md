# Ψ_universe Attractor Library v1.0

Modular Python implementation of the Ψ_universe attractor filter.

## Features
- Reusable PsiFilter class
- Automatic parameter optimizer
- PCA reduction to 3D
- Benchmark notebook

## Installation
pip install numpy scikit-learn scipy
